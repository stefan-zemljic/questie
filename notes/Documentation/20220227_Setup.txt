Git
    Install git-scm on my local windows-10 machine with default config

        git init
    Make the initial commit
    Creat the git repository: https://github.com/stefan-zemljic/questie.git
    Push to the repository
        git remote add origin https://github.com/stefan-zemljic/questie.git
        git branch -M main
        git push -u origin main
