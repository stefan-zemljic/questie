React / Typescript / Yarn 2
    Install yarn globally
        npm install -g yarn
    Initialize react project
        yarn create-react-app questie --template typescript
        copied content of ./questie to ./
    Set project yarn version to yarn 2
        yarn set version berry
    Install dependencies
        yarn install

Execute and test build
    yarn build
    npm install -g serve
    serve -s build

Start dev server - works with live reload
    yarn start
